# DayZ Интерактивная карта ChernarusPlus

Веб-приложение для работы с интерактивной картой DayZ ChernarusPlus с поддержкой пользовательских меток, поиска, фильтрации и экспорта/импорта.

! Импорт и Экспорт подогнан под метки lbmaster (lbgroup)

### Актуальная web-версия всегда есть тут: 
- [Черно](https://dayzmap.silthost.ru)
- [DeerIsle](https://deerisle.silthost.ru)
- [Deadfall](https://deadfall.silthost.ru)

### Полное руководство пользователя:
- https://dayzmap.silthost.ru/guide/user_guide.html

## 🆘 Поддержка

При возникновении проблем:
1. Проверьте консоль браузера на наличие ошибок (F12)
2. Убедитесь что все файлы загружены корректно
3. Проверьте доступность директорий с тайлами
4. Убедитесь что веб-сервер правильно обслуживает файлы
